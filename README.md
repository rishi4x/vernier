# Vernier

Vernier is a small macOS menu bar app for measuring pixels on screen.

## Install

1. Download `Vernier.dmg` from the [releases page](https://github.com/rishi4x/vernier/releases).
2. Open `Vernier.dmg`.
3. Drag `Vernier.app` into the `Applications` folder.
4. Open Vernier from `Applications`.

Vernier is currently unsigned, so macOS may block it the first time. To allow it:

1. Try to open `Vernier.app`.
2. When macOS shows the warning, click `Done`.
3. Open `System Settings` -> `Privacy & Security`.
4. Scroll to the security message about Vernier.
5. Click `Open Anyway`, then confirm with `Open`.

## Use

1. Launch Vernier.
2. Click the ruler icon in the menu bar.
3. Choose `Measure`, or set a shortcut from `Settings...`.
4. Move the cursor to see the horizontal and vertical distances to the nearest detected boundaries.
5. Click and drag to measure a rectangular region.
6. Press `Esc` to leave measurement mode.

Vernier needs Screen Recording permission so it can read the screen:

1. Open `System Settings`.
2. Go to `Privacy & Security` -> `Screen & System Audio Recording`.
3. Enable Vernier.
4. Restart the app if macOS asks.
